# Bootstrap 5 Learning Journey

This repository serves as a practical workspace for learning and mastering **Bootstrap 5**. My goal is to build responsive, mobile-first, and efficient front-end interfaces.

## 🛠 Covered Topics
- **Layout & Grid:** Implementing the grid system for responsive design.
- **Components:** Mastering core UI elements:
  - 🎨 **Colors:** Utilizing Bootstrap utility classes and color schemes.
  - 🔘 **Buttons:** Implementing various button styles and states.
  - ⚠️ **Alerts:** Creating interactive notification components.
  - 🃏 **Cards:** Building structured content cards.
  - 🖼 **Carousel Slider:** Implementing dynamic image sliders.
  - 🔽 **Dropdown:** Creating functional dropdown menus.

## 🚀 Technologies Used
- HTML5
- CSS3
- Bootstrap 5 (CDN)

## 💡 Future Goals
As a Backend Developer (Django), my objective is to integrate these front-end designs into **Django Templates** and manage them using Django's static files system.
